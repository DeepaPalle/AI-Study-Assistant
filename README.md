# 🤖 AI Study Assistant

An AI-powered study assistant built using Streamlit and Google Gemini.

## Features

- 📚 Learn any topic with AI explanations
- 🧩 Generate quizzes from topics
- 📄 Upload PDF notes
- ❓ Ask questions from PDFs
- 📝 Generate PDF summaries
- 🎯 Generate quizzes from PDFs
- 🗂️ Generate flashcards from PDFs
- 📈 View learning history

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- PyPDF2
- Scikit-Learn

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Study-Assistant.git
cd AI-Study-Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create .env File

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Application

```bash
streamlit run interactive_learning_buddy.py
```
